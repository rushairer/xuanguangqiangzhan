# 炫广枪战

Mind+ 项目转换后的独立 Web 游戏。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

Mind+ 原始项目文件和素材集中在 `mindplus-source/`，Web 运行时代码在 `src/`。


《炫广枪战》说明书
1.游戏目标
你在一个广场里，你可以派出新夜怪向别的人发起挑战
达成击败王队长、满级、血量达到46级、
攻速达到21级这四个条件即可通关并进入下轮回
2.游戏界面
此游戏有3个界面，
开局是主界面，
点击【挑战】可进入战斗界面
点击【新夜怪训练场】可进入训练场界面
3.游戏胜负判定
在战斗界面的白色进度条是血条
上面是玩家血条，下面是电脑血条
击败对手即可获胜
新夜怪训练场分为3个阶段：攻击阶段、闪避阶段、
 实战阶段
3.1.攻击阶段击败所有五角星即可获胜
3.2.闪避阶段躲五角星，躲100个
3.3.实战阶段击败对手获胜
4.按键介绍
4.1主界面按键
【1】升级血量【2】升级护盾防御【3】升级攻速
【4】升级子弹发射数量【5】查看技能点
【6】关闭技能点显示【7】打开战斗烦恼名单
【8】关闭战斗烦恼名单【9】打开邀请战斗名单
【10】关闭邀请战斗名单【r】查看道具【t】关闭道具
【y】显示列表并升级五大招式【u】关闭五大招式列表
【d】查看五大招式【f】关闭五大招式显示
【g】切换模式【h】查看优势统计图
【j】关闭优势统计图【k】添加好友【l】删除好友
【x】签到【上】查看玩家信息【下】关闭玩家信息
【左】搜索人物（可查看“7姓名编号”）【右】切换人物
4.2.战斗界面按键
【1】召唤dz【2】清除dz【3】、【4】切换dz
【空格】开枪【g】向上移动【v】向下移动
【c】向左移动【b】向右移动【z】+【空格】释放绝招
【q】使用持攻豆招式【w】使用金光薇招式
【e】使用毁灭毒针招式【x】使用炫夜护盾招式
【z】+【x】使用轰炸匕首招式【k】召唤好友
【n】打开其他招式列表，点击即可释放【m】蓄力
5.提示
5.1.如果觉得太难，可以按下【上】调节难度
5.2.当子弹不足时，【购物】会提示，小心突然没子弹
5.3.道具可以通过签到、打怪获得
6.特殊语术

qd=电脑首领，yb=玩家的兵，db=电脑的兵
